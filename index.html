Claude Code 配額風暴 — Max 用戶 1 小時耗盡額度、Anthropic 承認問題

Anthropic 公開承認用戶消耗 Claude Code 配額「遠超預期」。Reddit、GitHub 論壇湧入大量投訴：Max 5 用戶（$100/月）反映 1 小時就耗盡原本可用 8 小時的額度；Max 20x 用戶則報告單一 prompt 導致使用率從 21% 直接跳至 100%。根因涉及三重疊加：Anthropic 在平日尖峰時段（太平洋時間 5am-11am）刻意限縮五小時配額窗口、已確認的計量器去同步 bug、以及 3 月 2x 離峰優惠結束。影響約 7% 用戶。Anthropic 未公開具體限額數字，缺乏透明度引發社群強烈不滿，多名用戶表示正在評估退款與取消訂閱。分析師指出，隨著 vibe-coding 與 Computer Use 等 agentic 功能興起，所有 AI 供應商都面臨固定月費制與 token 爆量消耗之間的結構性矛盾。
